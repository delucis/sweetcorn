# `sweetcorn` docs site

This folder contains the source code for the Sweetcorn documentation site built with [Starlight](https://starlight.astro.build/).

## Documentation

Run the documentation locally by running the following command in your terminal:

```shell
pnpm run dev
```

Content can be found in the [`src/content/docs/`](./src/content/docs/) directory.
