# `@sweetcorn/astro`

A custom Astro image service to generate dithered images using Sweetcorn.

## Installation

```bash
npx astro add @sweetcorn/astro
```

## Documentation

Learn more in the [Sweetcorn documentation site](https://delucis.github.io/sweetcorn/).
